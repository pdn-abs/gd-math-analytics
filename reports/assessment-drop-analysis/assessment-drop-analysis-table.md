### **Tier 2 Action Plan for Assessment Level Improvement**

| Level ID | SkillAge | Reason: Why This is a Problematic Level (with Data Evidence) | Action To Be Taken |
| :--- | :--- | :--- | :--- |
| **`matchNumberCount1to72AL`** | 4 | **High-Volume "Early Quit":** This level has the second-largest drop count (**61 users**) of any "Early Quit" level. With a **46% drop rate** but only **5.6 wrong moves/user**, it strongly suggests a UX or comprehension barrier at the start of the level for a large number of players. | **Audit Intro/First Board:** As with the other "Early Quit" levels, review the initial instructions and board layout. This is a high-impact target after the DOB flow is fixed. |
| **`matchObjectWithFraction1AL`** | 10 | **Likely "Parent's Age" + Difficult Concept:** This level has a high volume (118 starters) and a high **42% drop rate**. | **`orderUpToSevenNumber2AL`** | 5 | **Comprehension Barrier:** This level has a **42% drop rate**, and the average time spent by users who drop is a very high **72 seconds**. This long duration before quitting, combined with a high success rate (**85%**) for completers, points to users getting stuck on the instructions and giving up out of frustration. | **Clarify Instructions:** Redesign the level's introduction to be more visual and less reliant on text. It will likely need simplification or better scaffolding. |
| **`stackTensAndOnes20UpTo502AL`** | 6 | **"Persisting Difficulty" (High Struggle, No Drops):** This level is a unique case. It has a **0% drop rate**, but the **second-highest wrong moves per user (12.0)** in the entire assessment. Users are determined to finish but are struggling significantly. | **Add Scaffolding/Hints:** The level is engaging but too hard. Implement in-level hints, visual aids, or break down the boards into simpler steps to reduce user struggle and improve the success rate. |
| **`matchEdgesCorners22AL`** | 6 | **"Persisting Difficulty" (High Struggle, Low Success):** Similar to the above, this level has a low drop rate (**9%**) but a very high **10.9 wrong moves/user** and a dismal **65% success rate** even for those who finish. The core concept is not being learned effectively. | **Redesign Core Mechanic:** The audio-only cue for this level is likely the cause of the low success rate. Add a visual icon or label to accompany the audio instruction to make the task clearer. |
